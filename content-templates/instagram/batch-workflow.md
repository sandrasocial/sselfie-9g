# Weekly Content Batching Workflow (2 hours/week)

## Sunday Evening (2 hours total):

### Hour 1: Create Content

**1. Generate 4 Post Images (30 minutes)**
- Use SSELFIE Feed Planner to create:
  - Monday: Behind-the-scenes image
  - Wednesday: Value post image/carousel
  - Friday: Transformation image/carousel
  - Sunday: Community post image

**2. Record 2-3 Reels (20 minutes)**
- Batch record all clips for the week
- Don't edit yet, just capture footage
- Use phone camera (vertical, 9:16 ratio)
- Record multiple takes of each clip

**3. Screenshot Customer Testimonials (10 minutes)**
- Save 3-5 customer messages (with permission)
- Screenshot user transformations
- Save any positive feedback

**4. Take BTS Photos for Stories (10 minutes)**
- Coffee + laptop shots
- Working moments
- Casual, authentic photos
- Save to phone camera roll

---

### Hour 2: Write Captions

**1. Write 4 Captions (30 minutes)**
- Use templates from:
  - `monday-behind-the-scenes.md`
  - `wednesday-value-posts.md`
  - `friday-transformations.md`
  - `sunday-community.md`
- Fill in brackets with specific details
- Add your personal voice/tone
- Include hashtags (5-10 per post)

**2. Write Reel Scripts (15 minutes)**
- Use scripts from `reels-scripts.md`
- Adapt to your style
- Note timing for each section
- Plan text overlays

**3. Plan Story Text Overlays (10 minutes)**
- Use templates from `instagram-stories-templates.md`
- Write text for each story
- Plan polls/questions
- Note which stories need link stickers

**4. Save Everything (5 minutes)**
- Save captions in Notes app or Google Doc
- Organize by day of week
- Label images clearly
- Create folder structure on phone

---

## During Week:

### Daily Tasks (10-15 minutes/day):

**Morning (9 AM):**
- Post scheduled feed post (if using Instagram scheduler)
- Post morning story with link sticker
- Engage with comments from previous day

**Midday (12 PM):**
- Post midday story (feature spotlight or poll)
- Check DMs and respond
- Engage with comments

**Afternoon (3 PM):**
- Post afternoon story (customer win or tip)
- Engage with comments
- Check analytics

**Evening (7 PM):**
- Post evening story (BTS or reflection)
- Final engagement check
- Plan tomorrow's stories

---

## Tools Needed:

**Content Creation:**
- SSELFIE Feed Planner (for post images)
- Phone camera (for reels and BTS)
- Canva or similar (for graphics if needed)

**Scheduling:**
- Instagram native scheduler (free)
- Or post manually at optimal times

**Organization:**
- Notes app (iPhone) or Google Docs (for captions)
- Phone camera roll (organized folders)
- Calendar reminder (for posting times)

**No Buffer needed!** Instagram's native scheduler works great for feed posts.

## Time-Saving Tips:

1. **Batch Everything:** Do all content creation in one 2-hour block
2. **Reuse Templates:** Don't reinvent the wheel - use templates
3. **Repurpose Content:** Turn top posts into reels, reels into carousels
4. **Save Responses:** Create saved replies for common DMs
5. **Set Reminders:** Use phone reminders for posting times
6. **Engage Efficiently:** Set 10-minute timer for daily engagement

---

## Monthly Review (1 hour/month):

**First Monday of each month:**
- Review analytics from previous month
- Identify top-performing content
- Note what didn't work
- Adjust templates based on data
- Plan monthly theme (if using)
- Set goals for new month

**Questions to Ask:**
- Which posts got most saves?
- Which reels got most shares?
- Which stories got most link clicks?
- What time of day performs best?
- What content type converts best?

**Optimize:**
- Do more of what works
- Remove what doesn't
- Test new formats
- Adjust posting schedule
