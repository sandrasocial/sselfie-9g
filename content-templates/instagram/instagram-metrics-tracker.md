## Key Metrics to Watch:

1. **Engagement Rate:** (Likes + Comments + Saves) / Followers

2. **Save Rate:** Saves / Impressions
   - Goal: 2-4%

3. **Link Click Rate:** Link clicks / Profile visits
   - Goal: 5-10%

4. **Conversion Rate:** Purchases / Bio clicks
   - Goal: 2-5%

5. **Story Completion Rate:** Completions / Starts
   - Goal: 70%+
